# Webタイポグラフィ必携用語集

## 概要
Webタイポグラフィーに関するキーワードを解説。
日本語でも普及し始めてきたWebフォントだが、英語圏に比べ、そのノウハウは、あまり蓄積されておらず、議論も少ない。
Webフォントサービスを提供するFONTPLUSによるサポートのもと、ウェブと日本語のあり方の議論するための場所をつくる。
