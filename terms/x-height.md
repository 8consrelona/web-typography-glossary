# エックスハイト

エックスハイト（x-height）は、xなどアセンダーを持たないアルファベットの小文字の、ベースラインから文字の上端までの高さをいいます。

![ラテンアルファベットの構造](../images/x-height.png)

## 実践

CSSではエックスハイトを基準にした単位`ex`が利用できます。1exがエックスハイトと等しくなります。

```css
p {
  line-height: 4ex;
}
```
