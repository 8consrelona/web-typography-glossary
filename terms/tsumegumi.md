# ツメ組み

ツメ組みは、字間を詰めてテキストを配置することです。

（*TODO* 図版）

## 実践

CSSの`letter-spacing`プロパティにマイナス値を指定することでツメ組みになります。

```css
h2 {
  letter-spacing: -0.05em;
}
```

ただし、ツメ組みは読みやすさを損なうことも少なくありません。採用には慎重になるべきでしょう。とくに長い文章では窮屈な印象になってしまうため避けた方が良いでしょう。
